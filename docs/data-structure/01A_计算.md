# 01A 计算

该章介绍了计算机科学的研究对象，计算。

## 绳索计算机

使用绳索，利用一定的算法，画直角三角形。

## 尺规计算机

使用尺规，利用一定的算法，将一个线段三等分。

## 算法

计算： 借助某种工具，按照某种规则，以明确而机械的形式进行

算法：

- 输入
- 输出
- 正确性，确定性，可行性

## 有穷性

hailstone序列，有增有减，目前无法确定是否是有穷的。
程序不一定等于算法，因为会溢出，报错等。

## 好算法

- 正确性
- 健壮
- 可读性 人与电脑，人与人沟通的
- 效率 时间，空间。算法 + 数据结构 = 程序(DSA)。对算法而言最重要的特性是效率。
